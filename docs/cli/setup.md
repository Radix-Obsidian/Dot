# `dot setup`

Initialize `~/.dot/dot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
dot setup
dot setup --workspace ~/.dot/workspace
```

To run the wizard via setup:

```bash
dot setup --wizard
```
